문제 1

다음의 속성을 가지는 사각형 객체를 만들어 본다.

속성명(한글) 속성명(영문) 설명
너비 width 사각형의 너비값을 가지는 속성이다.
높이 height 사각형의 높이값을 가지는 속성이다.
색상 color 사각형의 내부 배경색을 가지는 속성이다.
이 속성값은 색상코드 또는 색상명을 가진다.
면적 getArea 사각형의 너비와 높이로 계산된 면적이다.

객체를 생성할 때 속성명은 한글이 아닌 영문으로 작성해야 한다.
면적의 경우 함수로 만들어서 내부 속성값으로 계산되여 반환될 수 있게 한다.
너비, 높이, 색상 속성의 값은 input 태그에 입력된 값으로 설정되도록 한다.
최종 결과는 res6 영역에 정보를 출력하는 것으로 확인한다.
